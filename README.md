Ecoyaan Checkout Flow (Frontend Assignment)

This project implements a simplified checkout experience inspired by Ecoyaan.

Tech Stack
- Next.js
- React
- Tailwind CSS
- Context API
- Server Side Rendering

Features
- Cart page with SSR data fetching
- Shipping address form with validation
- Payment confirmation screen
- Order success page
- Mock API using Next.js API routes
- Responsive UI

Project Flow

Cart
→ Shipping Address
→ Payment
→ Order Success

Run Locally

npm install
npm run dev

Deployment

The project is deployed on Vercel.
